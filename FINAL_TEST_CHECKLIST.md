# そらギルド 最終テストチェックリスト

実運用前に、iPhone / iPad / PC で主要機能を確認するためのチェックリストです。

## 1. ホーム画面追加（PWA）

- [ ] iPhone Safariでアプリを開く
- [ ] 共有メニューから「ホーム画面に追加」を実行する
- [ ] ホーム画面に「そらギルド」として追加される
- [ ] 追加したアイコンから起動すると、ブラウザではなくアプリ風に表示される
- [ ] iPad / PCでも通常表示が崩れない

## 2. 初回オンボーディング

- [ ] 初回起動時に使い方ガイドが表示される
- [ ] 「次へ」で各スライドを進められる
- [ ] 「スキップ」で閉じられる
- [ ] 最後の「はじめる」で閉じられる
- [ ] 再読み込み後、オンボーディングが再表示されない

## 3. 親モードPIN

- [ ] ギルド管理タブを開くとPIN入力が表示される
- [ ] 間違ったPINでエラーメッセージが表示される
- [ ] 正しいPINで親モードに入れる
- [ ] 親モード中の表示が出る
- [ ] 「親モード解除」で通常モードに戻る
- [ ] 再読み込み後、親モードが解除されている

## 4. クエスト追加・編集・削除

- [ ] 親モードでクエスト追加フォームが表示される
- [ ] クエスト名 / 説明 / XP / Gold を入力して追加できる
- [ ] 種別 / 繰り返し / 曜日 / ステータス / 優先度 / カテゴリを設定できる
- [ ] 追加したクエストがクエスト画面に表示される
- [ ] 編集で内容を変更できる
- [ ] 削除前に確認ダイアログが表示される
- [ ] 削除後、クエスト画面から消える
- [ ] 再読み込み後も追加・編集内容が保持される

## 5. 毎日クエスト / チャレンジクエスト切り替え

- [ ] クエスト画面に「毎日クエスト」と「チャレンジ」タブが表示される
- [ ] タブをタップして切り替えられる
- [ ] スマホで左右スワイプして切り替えられる
- [ ] 毎日クエストには daily_required のクエストだけ表示される
- [ ] チャレンジには challenge のクエストだけ表示される
- [ ] チャレンジクエストが毎日クエスト達成率に含まれない

## 6. 曜日指定クエスト

- [ ] 親モードで「曜日指定」を選べる
- [ ] 複数曜日を選択できる
- [ ] 今日の曜日に該当するクエストだけ表示される
- [ ] 今日以外の曜日指定クエストは表示されない
- [ ] 曜日ラベル（月・水・金など）がカードに表示される
- [ ] 日付が変わった後、対象曜日の完了状態が正しくリセットされる

## 7. クエスト完了・取り消し

- [ ] 完了ボタンを押すと確認UIが表示される
- [ ] 「キャンセル」で何も変更されない
- [ ] 「はい」で完了処理が実行される
- [ ] 完了済みカードは再度押せない
- [ ] 親モード中のみ「未完了に戻す」が表示される
- [ ] 取り消し前に確認が表示される
- [ ] 取り消し後、XP / Gold / ステータスが戻る

## 8. 今日の分、達成！表示

- [ ] 毎日クエストが残っている場合、未完了数が表示される
- [ ] 毎日クエストをすべて完了すると「今日の分、達成！」が表示される
- [ ] チャレンジクエストが残っていても、毎日クエスト完了だけで達成扱いになる
- [ ] ホーム画面の「今日の任務」カードにも達成状態が反映される

## 9. XP / Gold / ステータス加算

- [ ] クエスト完了でXPが加算される
- [ ] クエスト完了でGoldが加算される
- [ ] 対応ステータスが +1 される
- [ ] XPが100ごとにレベルアップする
- [ ] XPバーが進む
- [ ] レベルアップ演出が表示される
- [ ] 再読み込み後も数値が保持される

## 10. 実績バッジ

- [ ] 成長画面に実績セクションが表示される
- [ ] 未達成実績が暗めに表示される
- [ ] 条件達成時に実績が解除される
- [ ] 実績解除トーストが表示される
- [ ] ホーム画面に最近解除した実績が最大3件表示される
- [ ] 再読み込み後も解除状態が保持される

## 11. ご褒美交換

- [ ] 親モードでご褒美を追加できる
- [ ] ご褒美を編集できる
- [ ] ご褒美を削除できる
- [ ] 子ども側で交換可能なご褒美が表示される
- [ ] Gold不足のご褒美は交換できない
- [ ] 交換前に確認メッセージが表示される
- [ ] 交換後、Goldが減る
- [ ] 交換履歴が残る

## 12. GASメール通知

- [ ] ご褒美交換時にGASへPOST送信される
- [ ] 通知成功時、交換処理が正常に完了する
- [ ] 通知失敗時も交換処理は止まらない
- [ ] 通知失敗時に軽いメッセージが表示される
- [ ] メール本文に名前 / ご褒美 / 消費Gold / 残りGold / 日時が表示される

## 13. 週間レポート送信

- [ ] 成長画面に週間レポートが表示される
- [ ] 今週のクエスト達成数 / XP / Gold / ステータス増加が表示される
- [ ] 親モード中のみ「週間レポートを送信」ボタンが表示される
- [ ] 手動送信でGASへPOSTされる
- [ ] 成功時に「週間レポートを送信しました」と表示される
- [ ] 失敗時に「送信に失敗しました」と表示される

## 14. スプレッドシート保存

- [ ] GAS側で `weekly_reports` シートが作成または再利用される
- [ ] 1行目に見出しが作成される
- [ ] 週間レポート送信時に1行追加される
- [ ] 保存日時 / 週の開始日 / 週の終了日が入る
- [ ] クエスト達成数 / XP / Gold / STR / INT / END / DEX / 連続ログイン日数が入る
- [ ] シートの見出し・罫線・列幅が見やすく整形される

## 15. バックアップ / 復元

- [ ] 親モードでバックアップ機能が表示される
- [ ] 「バックアップをダウンロード」でJSONファイルを保存できる
- [ ] 「バックアップを読み込む」でJSONファイルを選択できる
- [ ] 読み込み前に上書き確認が表示される
- [ ] 正しいJSONで復元できる
- [ ] 不正なファイルではエラーメッセージが表示される
- [ ] 復元後、クエスト / XP / Gold / 実績 / 履歴が反映される

## 16. スマホ幅の表示崩れ

- [ ] iPhone SE幅で文字がはみ出さない
- [ ] iPhone通常幅で下ナビ5項目が横並びに収まる
- [ ] iPad幅でカードが不自然に広がりすぎない
- [ ] ボタンのタップ領域が十分にある
- [ ] モーダルが画面内に収まる
- [ ] 背景上の文字が読みにくくない
- [ ] スクロール時に下ナビが操作を邪魔しない
- [ ] PC表示でも主要操作が問題なく行える
