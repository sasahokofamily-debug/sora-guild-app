# そらアプリ Web Static

ビルド不要で動かせる Web版の雛形です。

## 使い方

- `index.html` をブラウザで開く
- ローカルサーバーで配信する場合は `web-static/` をそのまま公開する
- iPhone Safari で開いて `ホーム画面に追加` すると PWA 風に使えます

## 含まれているもの

- ホーム / クエスト / 能力 / 保護者 の静的UI
- 保護者画面の簡易依頼作成
- `localStorage` を使った依頼の端末内保存
- PWA用 manifest
- service worker
- 仮アイコン

## 次に足すもの

- プレイヤー側での完了記録
- 保護者側の定番依頼編集
- 進捗と能力の自動反映
- 複数端末の同期
